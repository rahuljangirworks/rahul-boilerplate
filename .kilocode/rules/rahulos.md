# Kilocode Operating Context

This project uses RahulOS conventions.
- Read `AGENTS.md` first.
- Refer to `.agent/BOOTSTRAP.md`, `.agent/walkthrough/README.md`, `PROJECT-STATE.md`, and `PROJECT-CONTROLS.md`.
- Keep `.agent/plan/PROJECT-PACKET.md` and `.agent/plan/PHASES.md` parseable for the central approval portal.
- Approval evidence belongs in `.agent/approvals/`.
- Keep `PROJECT-STATE.md` as the source of truth for stage/current action.
